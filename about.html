Mission

About this project

This project's objective is to create a simple yet dynamic website for aquarium pet owners, both new and experienced. The development will first emphasize design and nagivation, then a database will be created in order to hold data for a future community.

This project will contain the following:

*A registration system for new users and existing users.

*Social media plug-ins.

*Highly responsive design.

*A sample shop page.



This project will be updated and modified with new and better features and a blog will be created to report these changes.(has not been created yet.
